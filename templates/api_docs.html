üìñ Overview

The HoloCrypt API provides endpoints for encrypting and decrypting messages using multiple cipher algorithms, sending encrypted emails with QR codes, and SMS notifications.


Base URL: http://localhost:5000/api/v1

Content-Type: application/json

üîë Authentication

Currently, the API is open for testing. In production, you should implement API key authentication.

üõ†Ô∏è API Endpoints

GET /api/v1/health

Check API health status

curl http://localhost:5000/api/v1/health

Response:

{ "status": "healthy", "service": "HoloCrypt API", "version": "1.0.0" }

POST /api/v1/encrypt

Encrypt a message using specified cipher

Request Body:

{ "message": "Your secret message", "cipher_type": "caesar", "cipher_params": {"shift": 5} }

cURL Example:

curl -X POST http://localhost:5000/api/v1/encrypt \ -H "Content-Type: application/json" \ -d '{ "message": "Hello World", "cipher_type": "caesar", "cipher_params": {"shift": 5} }'

Response:

{ "success": true, "encrypted_text": "Mjqqt Btwqi", "cipher_type": "caesar", "cipher_params": {"shift": 5}, "original_length": 11, "encrypted_length": 11 }

POST /api/v1/decrypt

Decrypt an encrypted message

Request Body:

{ "encrypted_text": "Mjqqt Btwqi", "cipher_type": "caesar", "cipher_params": {"shift": 5} }

Response:

{ "success": true, "decrypted_text": "Hello World", "cipher_type": "caesar", "cipher_params": {"shift": 5} }

POST /api/v1/send-encrypted-email Premium

Encrypt message and send via email with password-protected PDF

Request Body:

{ "message": "Your secret message", "receiver_email": "punithns26@gmail.com", "sender_name": "Alice", "cipher_type": "caesar", "cipher_params": {"shift": 5}, "pdf_password": "secret123", "receiver_phone": "+1234567890", "password_hint": "Your PDF password is: secret123" }

Response:

{ "success": true, "message": "Email sent successfully", "encrypted_text": "Mjqqt Btwqi", "email_status": {...}, "sms_status": {...} }

POST /api/v1/generate-qr

Generate QR code for encrypted message

Request Body:

{ "encrypted_text": "Mjqqt Btwqi", "cipher_type": "caesar", "cipher_params": {"shift": 5} }

Response: PNG image

GET /api/v1/ciphers

List all available cipher types and their parameters

curl http://localhost:5000/api/v1/ciphers

Response:

{ "success": true, "ciphers": [ { "type": "caesar", "name": "Caesar Cipher", "description": "Shifts each letter by a fixed number", "params": {...}, "example": {...} }, ... ], "total": 5 }
